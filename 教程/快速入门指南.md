# MiniAI记事本 - 快速入门指南

> 5分钟快速了解项目核心概念和开发方式

---

## 1. 项目一句话描述

MiniAI记事本是一个采用 **Neo-Brutalist设计风格** 的轻量级个人信息管理应用，整合新闻、笔记、财务管理三大功能。

---

## 2. 核心技术栈

```
Vue 3 + TypeScript + Vite
├── 状态管理: Pinia
├── 本地存储: IndexedDB (Dexie.js)
├── 样式: Tailwind CSS + 自定义Neo组件
├── 富文本: Tiptap
└── 图标: Material Symbols
```

---

## 3. Neo-Brutalist设计三板斧

| 特征 | CSS实现 |
|------|---------|
| **无圆角** | `border-radius: 0` |
| **粗边框** | `border: 4px solid white` |
| **硬阴影** | `box-shadow: 8px 8px 0 rgba(255,255,255,0.1)` |

**Hover效果**：`translate(-4px, -4px)` + 阴影增强

## 5. 核心数据模型

```typescript
// 笔记实体
interface Note {
  id: string
  title: string          // ≤50字符
  content: string        // ≤5000字符，支持HTML
  category_id: string    // 分类ID
  tags: string[]         // 标签数组
  
  // Neo风格属性
  card_type: 'text' | 'image' | 'task' | 'quote'
  card_color: 'yellow' | 'cyan' | 'white' | 'gray'
  
  is_archived: boolean
  is_pinned: boolean
  created_at: number
  updated_at: number
}
```

---

## 6. 数据流：Repository模式

```
Page组件 → Pinia Store → Repository → IndexedDB (Dexie)
```

**优势**：业务代码不直接操作IndexedDB，便于后期云迁移。

```typescript
// Store中使用Repository
const notes = await noteRepository.findAll()

// Repository封装Dexie操作
class NoteRepository {
  async findAll() {
    return db.notes.toArray()
  }
}
```
