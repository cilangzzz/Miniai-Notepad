# 功能点规格文档：KINETIC_NOTES 记事本应用

## 项目概述
**项目名称**: KINETIC_NOTES
**设计风格**: Neo-Brutalist (新粗野主义) / Editorial Editorial
**主题特点**: 高对比度、锐利边缘、4px粗边框、倾斜元素、暗色背景(#131313)、黄色(#FFD700)与青色(#007F7F)双强调色

---

## 功能模块总览

| 模块 | 页面 | 核心功能 |
|------|------|----------|
| 笔记管理 | my_notes | 笔记列表展示、筛选排序、快速创建 |
| 分类标签 | categories | 分类管理、标签云、统计信息 |
| 归档搜索 | archive_search | 搜索查询、归档管理、恢复功能 |
| 编辑器 | edit_note | 笔记编辑、标签关联、提醒设置 |

---

## 1. 笔记列表页 (my_notes)

### 1.1 顶部导航栏
- **品牌标识**: 倾斜字体的"KINETIC_NOTES"标题
- **菜单按钮**: 侧边栏展开/收起控制
- **搜索按钮**: 快速跳转搜索页
- **用户头像**: 个人资料入口

### 1.2 侧边栏导航 (桌面端)
- **分类列表**: Work、Personal、Ideas、Tasks
- **存储容量指示器**: 显示存储使用百分比
- **视觉反馈**: 当前选中分类高亮显示

### 1.3 笔记卡片网格
- **Bento Grid布局**: 响应式1/2/3列网格
- **卡片类型**:
  - 文本笔记卡片
  - 图片笔记卡片
  - 任务清单卡片
  - 引用/名言卡片
- **卡片信息**:
  - 分类标签（带旋转角度）
  - 创建/更新时间
  - 标题
  - 内容摘要
  - 附件指示图标
- **交互效果**:
  - Hover时卡片上移(-translate-y-1, -translate-x-1)
  - Neo-brutalist阴影效果

### 1.4 筛选与排序
- **Filter按钮**: 打开筛选面板
- **Sort By按钮**: 排序方式切换

### 1.5 创建笔记
- **FAB按钮**: 固定在右下角的大号添加按钮
- **视觉样式**: 黄色背景(#FFD700)、白色边框、Neo阴影

### 1.6 移动端底部导航
- 四个导航项: Notes、Tags、Archive、Settings
- 当前页面高亮显示（上移+阴影效果）

## 2. 分类标签页 (categories)

### 2.1 分类管理
- **分类列表展示**: Work、Personal、Ideas、Tasks
- **分类操作**:
  - Hover显示编辑/删除按钮
  - 点击进入分类详情
- **新建分类**: 底部"NEW CATEGORY"按钮

### 2.2 标签云
- **标签展示**: 多彩标签按钮，带旋转角度
- **标签颜色**: 黄色、青色、白色、深灰多种配色
- **交互效果**:
  - Hover时旋转归位
  - 点击激活/筛选
- **标签管理**:
  - 筛选按钮 (filter_list)
  - 排序按钮 (sort_by_alpha)
- **新增标签**: 底部拖放区域提示

### 2.3 统计面板
- **Total Tags**: 标签总数统计
- **Tagged Notes**: 已标记笔记数统计

## 3. 归档搜索页 (archive_search)

### 3.1 搜索功能
- **搜索框**: 大号输入框，占位提示"QUERY THE ARCHIVE..."
- **Execute按钮**: 执行搜索
- **视觉层次**: 背景偏移层 + 主输入层

### 3.2 筛选芯片
- **时间筛选**: All Time、Last 30 Days
- **分类筛选**: Work、Ideas
- **交互**: Hover变青色背景

### 3.3 归档结果展示
- **卡片状态**: 默认灰度+透明度70%
- **Hover恢复**: 灰度取消+透明度100%
- **恢复按钮**: unarchive图标
- **归档日期**: 显示归档时间戳
- **卡片类型**:
  - 纯文本卡片
  - 图片卡片
  - 数据可视化卡片（进度条）
  - 横跨两列的大卡片

### 3.4 归档容量指示
- **容量显示**: 84%使用率
- **侧边栏位置**: 桌面端左侧导航底部

### 3.5 空状态提示
- "End of Archive Search Results"
- 调整查询参数提示

## 6. 用户交互流程

### 6.1 创建笔记流程
1. 点击FAB添加按钮
2. 进入编辑页
3. 输入标题和内容
4. 选择分类和标签
5. 设置提醒（可选）
6. 点击保存

### 6.2 浏览笔记流程
1. 主页查看笔记网格
2. 使用侧边栏切换分类
3. 使用筛选/排序调整视图
4. 点击卡片查看详情

### 6.3 搜索归档流程
1. 点击搜索图标或Archive导航
2. 输入搜索查询
3. 使用筛选芯片细化
4. Hover卡片预览恢复状态
5. 点击恢复按钮

### 6.4 管理分类标签流程
1. 进入Tags页面
2. 查看分类列表
3. 编辑/删除分类
4. 管理标签云
5. 创建新分类或标签

## 7. 响应式设计

### 7.1 断点策略
- Mobile: < 768px
- Tablet/Desktop: ≥ 768px
- Large Desktop: ≥ 1024px

### 7.2 Mobile端差异
- 侧边栏隐藏（通过menu按钮展开）
- 显示底部导航栏
- 单列笔记网格
- 编辑器底部工具栏

### 7.3 Desktop端差异
- 固定左侧导航栏
- 多列网格布局
- 编辑器侧边信息栏完整显示
