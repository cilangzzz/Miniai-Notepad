## 1. 研发目标

基于产品需求文档（PRD v1.1），设计并实现：

- **核心架构**：Vue3 + TypeScript + Vite 前端架构
- **数据存储**：前端存储（IndexedDB），后期可转云
- **设计风格**：Neo-Brutalist 高对比度视觉系统
- **性能目标**：页面加载≤1秒，支持离线使用

---

## 2. 核心设计原则

### 2.1 数据层抽象
采用 **Repository模式** 实现存储无关性：
```
Repository接口 → IndexedDB实现 → (未来) 云端API实现
```

### 2.2 云迁移预留设计
所有实体预留以下字段：
- `cloud_id`: 云端唯一标识
- `sync_status`: 同步状态（local/synced/pending）
- `last_sync_time`: 最后同步时间
- `version`: 数据版本号（冲突解决）

### 2.3 Neo-Brutalist设计系统
- **色彩**：背景`#131313`、主色`#FFD700`、次色`#007F7F`
- **边框**：4px白色粗边框、0px圆角
- **阴影**：硬偏移阴影(8px offset)
- **交互**：Hover上移(-translate-y-1)

## 4. 技术栈概要

### 4.1 前端框架
- **Vue 3** + Composition API + `<script setup>`
- **TypeScript** 严格模式
- **Vite** 构建工具

### 4.2 状态管理
- **Pinia** 状态管理
- **IndexedDB** 本地数据持久化（通过Repository抽象）

### 4.3 UI组件
- **TailwindCSS** 样式系统（自定义Neo-Brutalist主题）
- **自定义组件库**（无现成Neo风格UI库）

### 4.4 前端存储
- **IndexedDB** 作为主存储（支持大数据量、结构化查询）
- **Dexie.js** IndexedDB封装库（简化API）

## 5. 数据架构概要

### 5.1 实体类型
```typescript
// 基础实体接口（云迁移预留）
interface BaseEntity {
  id: string            // UUID，本地生成
  cloud_id?: string     // 云端ID（同步后填充）
  sync_status: 'local' | 'synced' | 'pending'
  last_sync_time?: number
  version: number       // 数据版本
  created_at: number    // 创建时间戳
  updated_at: number    // 更新时间戳
}

// 笔记实体
interface Note extends BaseEntity {
  title: string
  content: string
  category: string      // Work/Personal/Ideas/Tasks
  tags: string[]
  card_type: 'text' | 'image' | 'task' | 'quote'
  card_color: 'yellow' | 'cyan' | 'white' | 'gray'
  is_archived: boolean
  is_pinned: boolean
  reminder?: number
}

// 收入实体
interface Income extends BaseEntity {
  amount: number
  source: string
  date: string
  note?: string
}

// 支出实体
interface Expense extends BaseEntity {
  amount: number
  category: string
  date: string
  note?: string
}
```

### 5.2 Repository抽象层
```typescript
// Repository接口定义
interface Repository<T extends BaseEntity> {
  findAll(): Promise<T[]>
  findById(id: string): Promise<T | null>
  create(entity: Omit<T, 'id' | 'created_at' | 'updated_at'>): Promise<T>
  update(entity: T): Promise<T>
  delete(id: string): Promise<void>
  
  // 云同步预留接口
  syncToCloud?(): Promise<void>
  syncFromCloud?(): Promise<void>
}

// IndexedDB实现
class IndexedDBRepository<T> implements Repository<T> {
  // 使用Dexie.js封装IndexedDB操作
}
```
